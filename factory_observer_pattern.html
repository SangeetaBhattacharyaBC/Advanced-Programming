Factory Pattern (Object Creation)

Explanation: The Factory Pattern lets you create objects without specifying their exact class.

When to use: Use it when you have multiple related objects to create and you want to delegate the instantiation logic.

üé® Shape Creator

A drawing program uses a Shape Factory to create various shapes (Circle, Square, Triangle) on demand.

Show Python Code 
class Shape:
    def draw(self):
        pass

class Circle(Shape):
    def draw(self):
        return "Drawing a Circle üü¢"

class Square(Shape):
    def draw(self):
        return "Drawing a Square üü¶"

class Triangle(Shape):
    def draw(self):
        return "Drawing a Triangle üî∫"

class ShapeFactory:
    def create_shape(self, shape_type):
        if shape_type == "circle":
            return Circle()
        elif shape_type == "square":
            return Square()
        elif shape_type == "triangle":
            return Triangle()

factory = ShapeFactory()
shape = factory.create_shape("circle")
print(shape.draw())

Observer Pattern (Notification System)

Explanation: The Observer Pattern allows multiple objects (observers) to get updates automatically when one object (subject) changes.

When to use: Use it when changes to one object should automatically notify and update other dependent objects.

üì¢ Weather Station

Displays get automatic updates whenever the weather data changes.

Show Python Code 
class WeatherStation:
    def __init__(self):
        self.subscribers = []
        self.temperature = 25

    def subscribe(self, observer):
        self.subscribers.append(observer)

    def notify(self):
        for sub in self.subscribers:
            sub.update(self.temperature)

    def set_temperature(self, temp):
        self.temperature = temp
        self.notify()

class Display:
    def update(self, temp):
        print(f"Updated Display: Temperature is {temp}¬∞C üå§Ô∏è")

station = WeatherStation()
display = Display()
station.subscribe(display)
station.set_temperature(30)

üíπ Stock Market Tracker

Investors get notified automatically when stock prices change.

Show Python Code 
class StockMarket:
    def __init__(self):
        self.investors = []
        self.price = 100

    def register(self, investor):
        self.investors.append(investor)

    def notify(self):
        for inv in self.investors:
            inv.update(self.price)

    def set_price(self, new_price):
        self.price = new_price
        self.notify()

class Investor:
    def update(self, price):
        print(f"Investor notified: New Stock Price = ¬£{price}")

market = StockMarket()
investor = Investor()
market.register(investor)
market.set_price(120)

üì± Chat App Notifications

Users receive instant alerts when a new message is posted.

Show Python Code 
class ChatGroup:
    def __init__(self):
        self.users = []

    def join(self, user):
        self.users.append(user)

    def notify(self, message):
        for user in self.users:
            user.update(message)

class User:
    def update(self, message):
        print(f"New message: {message} üí¨")

group = ChatGroup()
u1 = User()
group.join(u1)
group.notify("Hello everyone!")
